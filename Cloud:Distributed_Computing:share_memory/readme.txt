1)source code:
When executing my source code of share-memory, we need to compile it on Eclipse environment then get the class file. I set 5 parameters when executing the code. In linux system, we run the program like: ‘java, Terasort, datasize, size of buffer, datadirectory, number of buffer, thread number’. For example, ’java Terasort 10731520 16384 /desktop 3000 2’
